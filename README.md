# Roman Numeral Converter

This is a responsive and accessible Roman Numeral Converter built using **HTML**, **CSS**, and **JavaScript**. Users can input any number between **1 and 3999**, and it converts it to its equivalent **Roman numeral**.

## 🔗 Live Demo

👉 [View Live Project](https://workspace-ankit.github.io/Roman-Numeral-Converter/)

## ✨ Features

- 🎯 Converts numbers (1 to 3999) to Roman numerals
- 🔍 Validates input with helpful messages
- 📱 Fully responsive and mobile-friendly
- 🧠 Uses efficient Roman numeral conversion logic
- 🧪 Supports both button click and `Enter` key
- 🎨 Visual error and success feedback

## 🚀 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
